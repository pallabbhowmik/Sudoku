# Sudoku

A Windows Forms Sudoku puzzle game built with .NET 8.

## Features

- **Puzzle Generation** — Generates valid Sudoku puzzles with 5 difficulty levels: Very Easy, Easy, Medium, Hard, Very Hard
- **Game Timer** — Tracks elapsed time while solving
- **Hint System** — Reveals a random empty cell with the correct answer
- **Validation** — Check your solution at any time for correctness
- **Auto-Solve** — Instantly reveal the full solution
- **Remaining Counter** — Shows how many cells are left to fill
- **Visual Grid** — Thick borders highlight the 3×3 sub-grids
- **Input Validation** — Only accepts numbers 1–9, with error feedback
- **Win Detection** — Automatically congratulates you when the puzzle is solved

## How to Play

1. Select a difficulty level and click **Start Game**
2. Fill in empty cells with numbers 1–9
3. Use **Hint** if you get stuck
4. Click **Check** to validate your solution
5. Click **Solve** to reveal the answer
6. Click **New Game** to start over
